What this tool does
Procedure Scope: This procedure includes both start and end dates for calendars and schedule terms to review, organized by school.

Validation Logic: A school is validated if it has an active schedule record. If no active schedule is found, this will be reflected in the output.

Term Boundaries: * End Dates: Includes terms only at the end of a series (e.g., S2, T3, Q4).

Start Dates: Includes terms only at the beginning of a series (e.g., S1, T1, Q1).

Full Year (FY): This is always included in the review.

How to read the output of the tool
Status Tracking: The tool informs you when the process has begun and ended, ensuring you know that all schools were validated.

Date Processing: If run with both dates returned, a school is processed twice: end date results are shown first, followed by the start date.
